# 🏋️ Personal Workout Performance Analysis Dashboard (Power BI & Excel)

## 📌 Project Overview

This project is based on my personal workout data logs, where I tracked exercises, sets, reps, weight, and recovery status.
Using Excel and Power BI, I analyzed workout volume, strength progression, and consistency trends.
It also includes injury tracking to measure performance impact and recovery patterns.

---

## 🎯 Objective

The main goal of this project is to combine fitness tracking with data analytics by transforming raw workout data into meaningful insights for performance improvement.

---

## 📂 Dataset Information

The dataset contains:

* Date & Day
* Session Details
* Workout Status
* Condition Status
* Muscle Group & Sub Muscle
* Exercise Name
* Weight Category
* Set Number
* Weight Lifted
* Reps
* Volume
* Intensity Rating
* Notes

---

## 📊 Dashboard Insights

This project helps analyze:

* Workout volume progression
* Strength growth over time
* Muscle-wise training frequency
* Workout consistency
* Exercise performance comparison
* Injury impact analysis
* Recovery trend tracking
